# 📊 Dashboard de Performance e Insights de Vendas (Power BI)

Este projeto apresenta um **Dashboard de Performance e Insights de Vendas** desenvolvido em **Power BI** utilizando o dataset Superstore.

O objetivo do dashboard é analisar o desempenho das vendas, lucratividade, impacto de descontos e performance de produtos para apoiar **tomadas de decisão baseadas em dados**.

---

# 🎯 Problema de Negócio

Empresas precisam entender como suas vendas estão performando em diferentes dimensões para responder perguntas como:

- Quais categorias geram mais lucro?
- Quais produtos estão gerando prejuízo?
- Como os descontos impactam a lucratividade?
- Quais regiões apresentam melhor desempenho de vendas?

Este dashboard fornece uma visão clara e interativa para responder essas perguntas.

---

# 📈 Visão Geral do Dashboard

O projeto contém **duas páginas analíticas**.

---

# 1️⃣ Página: Performance de Vendas

Esta página apresenta uma visão geral do desempenho da empresa.

### Indicadores (KPIs)

- Receita Total
- Lucro Total
- Margem de Lucro

### Análises

- Evolução das vendas ao longo do tempo
- Vendas por região
- Vendas por categoria
- Top 10 produtos por receita
- Distribuição geográfica das vendas

![Dashboard Performance](images/dashboard-performance.png)

---

# 2️⃣ Página: Insights Estratégicos de Vendas

Esta página foca em análises mais estratégicas para geração de insights de negócio.

### Indicadores

- Crescimento de vendas YoY
- Lucro total
- Margem média de lucro

### Análises

- Relação entre desconto e lucro
- Categorias mais lucrativas
- Top 10 produtos com maior prejuízo
- Distribuição geográfica do lucro

![Dashboard Insights](images/dashboard-insights.png)

---

# 🔎 Principais Insights

A partir da análise dos dados podemos observar que:

- As vendas cresceram **41,4% em relação ao ano anterior**
- Descontos elevados tendem a **reduzir significativamente o lucro**
- Alguns produtos apresentam **prejuízo recorrente**
- A categoria **Technology apresenta maior geração de lucro**
- A lucratividade varia significativamente entre regiões
